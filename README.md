# 🌆 Sustainable Smart City Assistant

The **Sustainable Smart City Assistant** is an AI-powered platform designed to enhance smart urban governance, improve sustainability KPIs, and support citizen engagement. It leverages **IBM Watsonx Granite LLM**, **FastAPI**, and **Streamlit** to provide actionable insights, anomaly detection, policy summarization, forecasting, and real-time chat support.

---

## 🧠 Features

- 🗣️ **AI Chat Assistant** powered by Granite LLM
- 📈 **KPI Forecasting** for water, energy usage, and more
- 🚨 **Anomaly Detection** to flag abnormal environmental data
- 📝 **Sustainability Report Generation** (Markdown & PDF)
- 🌿 **Eco-Tips & Policy Summarization**
- 📬 **Citizen Feedback System**
- 🔁 **Real-time API Integration**

---

## ⚙️ Technologies Used

| Stack        | Description                          |
|--------------|--------------------------------------|
| 🧠 IBM Watsonx | Granite LLM for natural language tasks |
| 🧰 FastAPI     | Backend API routing and logic       |
| 🎨 Streamlit  | Interactive UI frontend              |
| 📦 Pinecone   | Vector store for document search     |
| 📊 Scikit-learn | Time-series forecasting & anomaly detection |
| 📄 FPDF/Markdown | Report generation in multiple formats |
